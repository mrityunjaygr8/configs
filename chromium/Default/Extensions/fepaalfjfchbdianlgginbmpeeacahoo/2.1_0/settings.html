Toolbox for Google Play Store™
About this extension

Toolbox for Google Play Store™ is an extension that enhances the Google Play Store™ with many different features, which can all be enabled or disabled below.

A quick settings panel can be accessed by clicking the Toolbox icon in your browser's toolbar.

New in :

- Removed popup from clicking on disabled APKMirror button

- Minor bug fixes

  • open_in_new APKMirror button

    This adds a button for APKMirror on app listings. When clicked, it opens the given application on APKMirror in a new tab. If the app has not been uploaded to APKMirror, the button will turn gray.
  • open_in_new Android Police button

    This adds a button for Android Police on app listings. When clicked, it searches for posts about the given application on Android Police.
  • open_in_new AppBrain button

    This adds a button for AppBrain on app listings. When clicked, it opens statistics about the given app on AppBrain's site.
  • info_outline Beta program info

    This adds information about a given app's beta program on its app listing. If the app has a beta program, a link will be added to either join or leave it.
    Note: Toolbox cannot automatically check for beta programs that first require joining a Google+ community or a Google Group.
Clear cache

Toolbox keeps a local cache of data from APKMirror (when the APKMirror button is turned on) in order to load results quicker. If you are experiencing bugs with the extension, you should try clearing the cache. This will not erase your settings.
