You're using Toolbox on Firefox for Android, but due to a Firefox bug, the shortcut to Toolbox's settings in the browser's main menu might not work.

Mozilla has already fixed this in Firefox 57, but if you're using an older version, there's another way you can open the settings menu.

When you open any app page on the Play Store site, a gray bar at the top will open Toolbox's settings when you tap on it. The bar says "Toolbox extension settings," you can't miss it!
